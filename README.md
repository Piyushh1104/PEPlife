# PEPlife: Repository of Peptide Half-life Data

Welcome to the official repository for **PEPlife**, a manually curated database of experimentally validated peptide half-life data. This resource was developed to support researchers working in peptide therapeutics, bioinformatics, computational biology, drug discovery, and pharmacokinetics.

Web Server: http://crdd.osdd.net/raghava/peplife

# About the Database

PEPlife is a manually curated repository containing experimentally determined half-life information for therapeutic peptides. The database compiles scattered literature related to peptide stability, modifications, and pharmacokinetics into a centralized platform.

The database integrates information collected from:

* PubMed research articles
* Patents
* DrugBank
* Published experimental studies

PEPlife provides detailed information about peptide sequences, structures, biological activities, modifications, assays, and half-life values.

---

# Key Features

## Large Curated Dataset

* 2229 total entries
* 1193 unique peptides
* Data curated from:
  * 335 research articles
  * 16 patents
  * DrugBank peptide entries

---

## Comprehensive Annotations

Each database entry contains:

* Peptide name
* Peptide sequence
* Peptide length
* Half-life value
* Biological activity
* Assay information
* N-terminal modification
* C-terminal modification
* Chemical modifications
* Linear or cyclic configuration
* Chirality information
* Organism and media
* Patent ID

---

## Structural Information

PEPlife includes structural annotations obtained using:

* Protein Data Bank (PDB)
* PEPstrMOD
* I-TASSER
* DSSP secondary structure analysis

The database also provides:

* Predicted tertiary structures
* Secondary structure information
* SMILES representations

# Analysis Tools

* BLAST

These tools allow users to perform:

* Sequence similarity searches
* Structure analysis

* Structural alignment

# Peptide Modifications Included

PEPlife contains detailed information on peptide stability enhancement techniques such as:

* PEGylation
* Cyclization
* D-amino acid incorporation
* N-terminal modifications
* C-terminal modifications
* Amidation
* Acylation
* Glycosylation
* Fatty acid conjugation
* Non-natural amino acid substitutions

These modifications are important for improving peptide stability and bioavailability.
